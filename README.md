# 🎓 MOS Practice V15

MOS Practice V15 là phần mềm hỗ trợ luyện tập Microsoft Office Specialist (MOS) với khả năng tải dữ liệu từ Google Drive, tự động lưu cache và hoạt động ngay cả khi mất kết nối Internet.

---

## ✨ Tính năng

- 📥 Tải Questions từ Google Drive
- 📄 Tải Project Word từ Google Drive
- 💾 Lưu Cache tự động
- 🌐 Hỗ trợ Offline
- 🔄 Dễ dàng cập nhật dữ liệu mà không cần build lại phần mềm
- ⏱️ Đồng hồ đếm thời gian luyện tập
- 📊 Quản lý nhiều Project

---

## ⚙️ Yêu cầu

- Python 3.11 hoặc mới hơn
- Microsoft Word
- Windows 10/11

---

## 🚀 Cài đặt

Cài đặt thư viện:

```bash
pip install -r requirements.txt
```

---

## ▶️ Chạy chương trình

```bash
python main.py
```

Dữ liệu được lưu trên Google Drive:

- config.json
